Synerdy is an Android-based E-Commerce Application that can be used by buyers and sellers to facilitate their nerdy shopping experience! You can purchase products from an extensive list of categories from signed comics to CGC Graded books.

Superb Features

Multi Vendor Marketplace

Frontend Dashboard for Each Seller. Vendor Capabilities Management. Admin can manage new product status.

User Features

Customer can select any desired products from various categories.

Seller Features

Seller can add as many products as they like. Seller can delete the products that have sold.

Admin Features

Besides approving products that have been posted by sellers, the main use of the Admin Panel is to check if any unusual orders/products are added to the database.
